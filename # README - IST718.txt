Project Title: Airfare Prediction and Optimization using PySpark  
Course: IST 718 - Big Data Analytics

Project Description:
This project predicts airfare prices and proposes dynamic optimization strategies using large-scale airline datasets. The project applies PySpark for big data processing, feature engineering, model development (GBTRegressor), and evaluates model performance.

Folder Contents:
- Airfare Prediction and Optimization Report.pdf: Project report.
- AirfarePrediction_SparkCode.ipynb: PySpark Jupyter Notebook with full pipeline.
- AirfareOptimizationSlides.pptx: Summary presentation slides.

Software Requirements:
- Python 3.x
- Jupyter Notebook
- Apache Spark (PySpark)
- Libraries: pyspark, pandas, matplotlib, seaborn

Notes:
- Full modeling pipeline (preprocessing, feature extraction, prediction) is in PySpark.
- No separate raw dataset provided; analysis is fully documented in notebook.
